# CellularMax
Cellular automata rules simulator written in typescript

Rules implemented:
  - game of life
  - more coming...

## How to run

1. clone or download

2. `npm install`

3. `npm run build`

4. open index.html

5. click on grid to add or remove active cells

6. press Space to start/stop the simulation or press N to advance one cycle

### Tests

`npm run test`
